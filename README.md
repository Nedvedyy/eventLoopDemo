# Event Loop playground
This is for u to play for fun to understand more about how event loop works. Event queues,
  - timeout queues
  - I/O queues
  - check queues
  - close handler queues

  Two special queues
  - next tick queue
  - micro tasks queue

  Promises
  - native
  - Q
  - Bluebird

## Installation and run
Install the plugin from npm
```bash
$ npm install
$ npm run start
```
